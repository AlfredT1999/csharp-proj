DEFAULT VALUES

ALL TYPE INSTANCES HAVE A default VALUE. THE default VALUE FOR THE PREDEFINED TYPES IS THE RESULT OF A BITWISE ZEROING OF MEMORY:

1. REFERENCE TYPES (AND NULLABLE VALUE TYPES) -> null
2. NUMERIC AND ENUM TYPES -> 0
3. char type -> '\0'
4. bool type -> false

- TO GET THE default VALUE FOR ANY TYPE USE = default KEYWORD:

Console.WriteLine (default(decimal)); // 0

- YOU CAN OPTIONALLY OMIT THE TYPE WHEN IT CAN BE INFERRED:

decimal d = default;

PARAMETERS

A METHOD MAY HAVE A SEQUENCE OF PARAMETERS. YOU CAN CONTROL HOW PARAMETERS ARE PASSED WITH THE ref, in, AND out modifiers:

1. MODIFIER-(NONE) -> PASSED BY-(VALUE) -> GOING IN
2. MODIFIER-(ref) -> PASSED BY-(REFERENCE) -> GOING IN
3. MODIFIER-(in) -> PASSED BY-(READ-ONLY) -> GOING IN
4. MODIFIER-(out) -> PASSED BY-(REFERENCE) -> GOING OUT
