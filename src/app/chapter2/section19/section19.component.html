REF LOCALS

A SOMEWHAT ESOTERIC FEATURE OF C# IS THAT YOU CAN DEFINE A LOCAL VARIABLE THAT REFERENCES AN ELEMENT IN AN ARRAY OR FIELD IN AN OBJECT.

int[] numbers = {{ squareBraketOpen }} 0, 1, 2, 3, 4 };
ref int numRef = ref numbers[2];

// numRef IS A REFERENCE TO numbers[2]. WHEN WE MODIFY numRef, WE MODIFY THE ARRAY ELEMENT:

numRef *= 10;
Console.WriteLine(numRef); // 20
Console.WriteLine(numRef[2]); // 20

REF RETURNS

YOU CAN RETURN A REF LOCAL FROM A METHOD. THIS IS CALLED A REF RETURN.

using System;

public class Program
{{ squareBraketOpen }}
  static string x = "Old value";
  static ref string GetX() => ref x;// This method returns a ref.

  public static void Main()
  {{ squareBraketOpen }}
   ref string xRef = ref GetX(); // Assing result to a ref local.

   xRef = "New value";
   Console.WriteLine(x); // New value
  }
}
