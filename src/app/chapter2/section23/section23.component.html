# LOCAL VARIABLES

THE SCOPE OF A LOCAL VARIABLE OR LOCAL CONSTANT EXTENDS THROUGHOUT THE CURRENT BLOCK.

# SELECTION STATEMENTS

C# HAS THE FOLLOWING MECHANISM TO CONDITIONALLY CONTROL THE FLOW OF PROGRAM EXECUTION:

1. SELECTION STATEMENTS (IF, SWITCH)
2. CONDITIONAL OPERATOR (?:)
3. LOOP STATEMENTS (WHILE, DO-WHILE, FOR, FOREACH)

# THE IF/ELSE STATEMENT

AN IF STATEMENT EXECUTES A STATEMENT IF A BOOL EXPRESSION IS TRUE:

if(5 < 2 * 3) Console.WriteLine("true");

AN IF STATEMENT CAN OPTIONALLY FEATURE AN else CLAUSE:

else Console.WriteLine("false");

# THE SWITCH STATEMENT

SWITCH STATEMENTS LET YOU BRANCH PROGRAM EXECUTION BASED ON A SELECTION OF POSSIBLE VALUES THAT A VARIABLE MIGHT HAVE:

void ShowCard(int cardNumber)
{{squareBraketOpen}}
  switch(cardNumber)
  {{squareBraketOpen}}
   case 1:
    Console.WriteLine("Hi");
    break;// Jumps to the end of the switch statement
   case 2:
    Console.WriteLine("there");
    break;
   case 3:
    Console.WriteLine("!");
    break;
   case -1:
    goto case 1; // Jumps to another case clause. Another options is: goto default which jumps to the default clause.
   default:
    Console.WriteLine("No greetings!");
    break;
  }
}

WHEN MORE THAN ONE VALUE SHOULD EXECUTE THE SAME CODE, YOU CAN LIST THE COMMON CASES SEQUENTIALLY:

  switch(cardNumber)
  {{squareBraketOpen}}
   case 1:
   case 2:
   case 3:
   case 4:
    Console.WriteLine("Hi there!");
    break;
   default:
    Console.WriteLine("No greetings!");
    break;
  }

THIS FEATURE OF A SWITCH STATEMENT PRODUCES CLEANER CODE THAN MULTIPLE if-else STATEMENTS.
