TYPE BASICS

A VARIABLE DENOTES A STORAGE LOCATION THAT CAN CONTAIN DIFFERENT VALUES OVER TIME. IN CONTRAST, A CONSTANT ALWAYS REPRESENT THE SAME VALUE:

const int y = 360;

ALL VALUES IN C# ARE INSTANCES OF A TYPE.

PREDEFINED TYPE

PREDEFINED TYPES ARE TYPES THAT ARE SPECIALLY SUPPORTED BY THE COMPILER. FOR EXAMPLE, THE int TYPE IS A PREDEFINED TYPE FOR REPRESENTING A SET OF INTEGERS THAT FIT INTO 32 BITS OF MEMORY. FOR EXAMPLE:

int variable_1 = 12;
