# NESTED USING DIRECTIVES

YOU CAN NEST A USING DIRECTIVE WITHIN A NAMESPACE. THIS ALLOWS YOU TO SCOPE THE USING DIRECTIVE WITHIN A NAMESPACE DECLARATION.

namespace N1
{{squareBraketOpen}}
   class Class1 {{squareBraketOpen}}}
}

namespace N2
{{squareBraketOpen}}
   using N1;
   class Class2 {{squareBraketOpen}}} : Class1 {{squareBraketOpen}}}
}

namespace N2
{{squareBraketOpen}}
   class Class3 {{squareBraketOpen}}} : Class1 {{squareBraketOpen}}} // Compile-time error
}

# ALIASING TYPES AND NAMESPACES

IMPORTING A NAMESPACE CAN RESULT IN TYPE-NAME COLLISION. RATHER THAN IMPORTING THE ENTIRE NAMESPACE, YOU CAN IMPORT JUST THE SPECIFIC TYPES THAT YOU NEED, GIVING EACH TYPE AN ALIAS:

using PropertyInfo2 = System.Reflection.PropertyInfo;
class Program {{squareBraketOpen}} PropertyInfo2 p; }

AN ENTIRE NAMESPACE CAN BE ALIASED, AS FOLLOWS:

using R = System.Reflection;
class Program {{squareBraketOpen}} R.PropertyInfo2 p; }
