DEFAULT ELEMENT INITIALIZATION

THE DEFAULT ELEMENT FOR EACH ELEMENT WILL BE 0:

int[] a = new int[1000];
Console.Write(a[123]); // Equals to 0

* WHEN THE ELEMENT TYPE IS A VALUE TYPE, EACH ELEMENT VALUE IS ALLOCATED AS PART OF THE ARRAY, AS SHOWN:

// VALUE-TYPE:
Point[] a = new Point[1000];
int x = a[500].X; // 0

public Struct Point {{ squareBraketOpen }} public int X, int Y; }

* HAD POINT BEEN A CLASS, CREATING THE ARRAY WOULD HAVE MERELY ALLOCATED 1,000 NULL REFERENCES:

// REFERENCE-TYPE:
Point[] a = new Point[1000];
int x = a[500].X; // RUNTIME ERROR, NullReferenceException

public class Point {{ squareBraketOpen }} public int X, int Y; }

TO AVOID THIS ERROR, WE MUST EXPLICITLY INSTANTIATE 1,000 Points after instantiating the array:

for(int i = 0; i < a.Length; i++) {{ squareBraketOpen }} a[i] = new Point(); }

AN ARRAY ITSELF IS ALWAYS A REFERENCE TYPE OBJECT, REGARDLESS OF THE ELEMENT TYPE. FOR INSTANCE THE FOLLOWING IS LEGAL:

int[] a = null;
