INDICES AND RANGES

# INDICES:

SIMPLIFY WORKING WITH ELEMENTS OR PORTIONS OF AN ARRAY. INDICES LET YOU REFER TO ELEMENTS RELATIVE TO THE END OF AN ARRAY, WITH THE ^ OPERATOR:

char[] vowels = new char[] {{squareBraketOpen}} 'a','e','i','o','u'};
char lastElement = vowels [^1]; // 'u'
char secondToLast = vowels [^2]; // 'o'

C# IMPLEMENTS INDICES WITH THE HELP OF THE Index type:

Index first = 0;
Index last = ^1;
char firstElement = vowels [first]; // 'a'
char lastElement = vowels [last]; // 'u'

# RANGES:

RANGES LET YOU SLICE AN ARRAY BY USING THE .. OPERATOR:

char[] firstTwo = vowels [..2]; // 'a', 'e'
char[] lastThree = vowels [2..]; // 'i', 'o', 'u'
char[] middleOne = vowels [2..3]; // 'i'

C# IMPLEMENTS RANGES WITH THE HELP OF THE Range TYPE;

Range firstTwoRange = 0..2;
char[] firstTwo = vowels [firstTwoRange]; // 'a', 'e'
