NUMERIC LITERALS

AN UNDERSCORE CAN BE INSERTED ANYWHERE WITHIN A NUMERIC LITERAL TO MAKE IT MORE READABLE:

int million = 1_000_000;

NUMERIC SUFFIXES

NUMERIC SUFFIXES EXPLICITLY DEFINE THE TYPE OF A LITERAL:

float variable_1 = 1.0f;
// or:
float variable_1 = 1.0F;

THE F AND M SUFFIXES ARE THE MOST USEFUL AND SHOULD ALWAYS BE APPLIED WHEN SPECIFYING float OR decimal LITERALS.

STRING INTERPOLATION

A STRING PRECEDED WITH THE $ CHARACTER IS CALLED AN INTERPOLATED STRING. INTERPOLATED STRINGS CAN INCLUDE EXPRESSIONS ENCLOSED IN BRACES:

int x = 4;
Console.Write($"A square has {{ openBrace }}x} sides");

STRING COMPARISONS

string does not support < (less than) and > (major than) operators for comparisons. You must use the string's CompareTo() method.
