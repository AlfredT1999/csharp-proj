THE in MODIFIER

AN in PARAMETER IS SIMILAR TO A ref PARAMETER EXCEPT THAT THE ARGUMENT'S VALUE CANNOT BE MODIFIED BY THE METHOD (DOING SO GENERATES A COMPILE-TIME ERROR).

THE params MODIFIER

ALLOWS THE METHOD TO ACCEPT ANY NUMBER OF ARGUMENTS OF A PARTICULAR TYPE:

int total = Sum(1, 2, 3, 4);

Console.WriteLine(total); // 10

// The call to Sum above is equivalent to:
int total2 = Sum(new int[] {{squareBraketOpen}} 1, 2, 3, 4 });

int Sum(params int[] ints)
{{ squareBraketOpen }}
  int sum = 0;
  for (int i = 0; i < ints.Length; i++) sum += ints[i]; // Increase sum by ints[i]

  return sum;
}
